# 🤖 AI-Based Autonomous Navigation System

## 📌 Overview
This project simulates an AI-based autonomous navigation system using the A* pathfinding algorithm.

## 🎯 Problem Statement
Autonomous systems need to navigate efficiently while avoiding obstacles. This project demonstrates a solution using simulation.

## 🛠 Tech Stack
- Python
- Pygame

## ⚙️ Features
- Grid-based environment
- A* pathfinding
- Obstacle avoidance
- Real-time simulation

## 📂 Folder Structure

│── src/

## ▶️ How to Run

pip install pygame
python main.py
